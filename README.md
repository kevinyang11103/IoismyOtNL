# 社团管理系统 java27

## 项目概述

社团管理系统是一个基于Web的社团活动与信息管理平台。该系统实现了前后端分离，采用了Java和SpringBoot作为后端技术栈，以及Vue作为前端技术栈，致力于为社团提供一个便捷、高效的管理环境。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows
- **开发环境**: 
  - IDEA
  - JDK 1.8
  - Maven
  - Mysql 8.0
- **技术栈**: 
  - Java
  - Mysql
  - SpringBoot
  - Mybatis
  - Vue

## 功能模块

### 用户角色

- **用户模块**
  - 登录用户
  - 注册用户
  - 用户信息管理

- **超级管理员**
  - 拥有所有社团管理的权限

- **社团管理员**
  - 本社团信息管理
  - 社管活动管理
  - 入团申请
  - 费用管理
  - 社团活动通知

- **学生用户**
  - 社团信息查看
  - 社团活动参与
  - 入团申请
  - 费用支付

## 系统角色

系统定义了以下角色：

- 用户
- 超级管理员
- 社团管理员
- 学生用户
